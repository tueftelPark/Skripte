Alle aktuellen Skripte

Für initiales Setup, [FULL_RESET Skript hier downloaden](FULL_RESET.bat).

Dieses All-in-One Skript lädt die neueste Version von Arduino herunter, Installiert alle relevanten Arduino-Libraries am richtigen Ort und erstellt Verknüpfungen zu www.tinkercad.com und www.tuefteln.com/feedback auf dem Desktop.

Vor jedem Kurs kann das abgespeckte [Setup Skript hier downloaden](SETUP.bat) ausgeführt werden um nur den Desktop zurückzusetzen und die Libraries zu installieren.
